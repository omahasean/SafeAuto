# Readme

To run the file use github to import the file. 


If you would like to use your own CSV file navigate to the directory where you downloaded the repository. Drag and drop the csv file you would like to use into the main folder (SafeAutoKata/)

To run:
After the file is imported cd to the filepath and into SafeAutoKata/target/

once there run the command 
java -jar SafeAutoKata-1.0.jar

that should begin the command line program, there should be directions for how to use the program in the command line.
